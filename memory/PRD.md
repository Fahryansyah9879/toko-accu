# Toko Accu Prima Battery - Website PRD

## Project Overview
Modern, professional, mobile-friendly website for a car battery store in Cikarang, Bekasi, Indonesia.

## Business Information
- **Name:** Toko Accu Prima Battery
- **Category:** Toko Aki Mobil (Car Battery Store)
- **Rating:** 4.4★
- **Address:** Jl. Raya Cikarang - Cibarusah, Sukaresmi, Cikarang Selatan, Kabupaten Bekasi, Jawa Barat 17530
- **Google Maps Code:** M48C+7C Sukaresmi
- **Phone/WhatsApp:** 0821-5854-4469
- **Hours:** Daily until 21:00
- **Service:** Delivery available (Pesan Antar)

## User Personas
1. **Car Owners in Cikarang** - Need battery replacement, looking for trusted local store
2. **Motorcycle Owners** - Looking for affordable motorcycle batteries
3. **Emergency Cases** - Car broke down, need immediate battery replacement with delivery

## Core Requirements (Static)
- Indonesian language only
- No admin panel (static content)
- Example products and testimonials
- WhatsApp floating button integration
- Google Maps embed
- Mobile-first responsive design

## Implemented Features (March 2025)
### Pages
1. **Home (Beranda)** - Hero section, services, brands, CTA
2. **About (Tentang Kami)** - Store story, values, mission/vision
3. **Products (Produk)** - Car & motorcycle batteries catalog with WhatsApp order
4. **Services (Layanan)** - 6 services with FAQ accordion
5. **Reviews (Ulasan)** - Customer testimonials with 4.4 rating display
6. **Contact (Kontak)** - Contact info, Google Maps, quick action buttons

### Components
- Sticky navigation with mobile sheet menu
- WhatsApp floating button with pulse animation
- Footer with contact info and quick links
- Product cards with "Pesan via WA" buttons

### Design
- Color palette: Navy Blue (#1D3557), Emergency Red (#E63946), Voltage Yellow (#FFB703)
- Typography: Barlow Condensed (headings), Manrope (body)
- Mobile-first responsive layout
- Automotive/industrial style

### Tech Stack
- React (Frontend)
- FastAPI (Backend - not used for this static site)
- Tailwind CSS + Shadcn UI components
- Google Maps Embed API
